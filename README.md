# 🩺 Doctor Assistant (AI Medical Receptionist)

This is a **production prototype** of an AI-powered Medical Clinic Receptionist & Appointment System.

It is built as a **full-stack application** with:
✔ Web Chat interface for patients  
✔ AI-assisted appointment requests  
✔ Dashboard for doctors to manage appointments  
✔ Secure login for clinic staff

**Live demo:** https://doctor-assistant.lovable.app

---

## 🚀 Features

### 🧑‍⚕️ For Patients
- Chat with AI to request a clinic appointment
- Natural language input (Hinglish/English)
- Quick feedback and confirmation messages

### 🖥️ For Doctors / Staff
- Secure login for dashboard
- View pending appointment requests
- Approve or reschedule appointments
- See history of booked appointments
- Easy UI with minimal setup

---

## 🧠 How It Works

1. A patient opens the chat page
2. AI collects:
   - Patient name
   - Contact number
   - Preferred date/time
3. An **appointment request** is created and stored
4. Doctor logs into dashboard
5. Doctor approves, reschedules, or rejects appointment
6. Patient gets confirmation message (planned in future)
